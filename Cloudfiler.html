Default settings
NOTE: These settings will be applied to all new users
Upon Filing a message:
Sent messages:
Move to 'Filed' folder
  • Leave message in its folder
  • Move to 'Filed' folder
  • Move to 'Deleted Items' folder
Other messages:
Move to 'Filed' folder
  • Leave message in its folder
  • Move to 'Filed' folder
  • Move to 'Deleted Items' folder
Add filed date to Subject
Checked behaviour:
The date is added to the message subject line in the format [Filed on 2038-05-23]
Unchecked behaviour:
The subject line is not changed
Add CloudFiler category flag
Checked behaviour:
Messages are marked with a CloudFiler category Filed to CloudFiler
Unchecked behaviour:
No category is added
When drafting an email:
Automatically select the suggested filing location
Checked behaviour:
CloudFiler will automatically select the top-most suggested filing location
Unchecked behaviour:
Emails are only filed when the user selects a filing location
